# 检测更新失败

  1. 网络未正常连接
  2. 服务器关闭 解决方案如下：
  3. 正确连接网络后重试
  4. 使用浏览器打开更新地址：https://tatsuki-mengchen.lanzoub.com/b02unctwjopen in new window
    提取码：mengchen

# 提示settings.lua报错

如果您遇到了类似的情况，可能的原因如下：

  1. 软件未获得储存权限
  2. 配置文件出错 解决方案如下：
  3. 在设置中为软件开启储存权限
  4. 删除默认储存目录下/devToolbox/config.cfg配置文件
  5. 手动修复/devToolbox/config.cfg配置文件（不推荐）

# 无法正常使用脚本编辑器、开发者学院等WebView

如果您遇到了类似的情况，可能的原因如下：

  1. Android System WebView被手机厂商阉割
  2. Android System WebView版本太低，无法正常加载网页 解决方案如下：
  3. 下载最新版的Android System WebView（强烈推荐）
