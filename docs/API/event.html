# 事件：Event

事件系统是用脚本做玩法的基础。

local function gameStart(e)
    --添加内容
end
--监听“游戏开始”事件，事件发生时执行函数gameStart
ScriptSupportEvent:registerEvent([=[Game.Start]=], gameStart)

# 游戏逻辑事件：Game

名称

描述

参数

Game.Load

游戏加载中

无参数

Game.Start

游戏启动

无参数

Game.Run

游戏运行中

无参数，每0.05秒触发一次

Game.Hour

游戏内时间到整点时

hour：当前游戏内时间的小时

Game.RunTime

游戏运行了某秒时

ticks：游戏运行了多少帧

second：游戏运行了多少秒

Game.TimeOver

游戏超时

无参数

Game.End

游戏结束

无参数

Game.AnyPlayer.ReadStage

有玩家读取进度

无参数

Game.AnyPlayer.EnterGame

有玩家进入游戏

eventobjid：触发事件的玩家迷你号

Game.AnyPlayer.Victory

有玩家获胜

eventobjid：触发事件的玩家迷你号

Game.AnyPlayer.Defeat

有玩家失败

eventobjid：触发事件的玩家迷你号

Game.AnyPlayer.LeaveGame

有玩家离开游戏

eventobjid：触发事件的玩家迷你号

# 方块事件：Block

名称

描述

参数

Block.Add

方块被创建

blockid：被创建的方块id

x,y,z：被创建方块的坐标

Block.Remove

方块被移除

blockid：被移除的方块id

x,y,z：被移除方块的坐标

Block.PlaceBy

方块被放置

eventobjid：放置方块的玩家迷你号

blockid：被放置的方块id

x,y,z：被放置方块的坐标

Block.DestroyBy

方块被破坏

eventobjid：破坏方块的玩家迷你号(被爆炸破坏的没有此参数)

blockid：被破坏的方块id

x,y,z：被破坏方块的坐标

Block.Trigger

方块被触发

(包括箱子被打开、炸药桶被点燃等等事件)

eventobjid：触发方块的玩家迷你号

blockid：被触发的方块id

x,y,z：被触发方块的坐标

Block.Fertilize

方块被施肥

eventobjid：施肥的玩家迷你号

blockid：被施肥的方块id

x,y,z：被施肥方块的坐标

Block.Dig.Begin

开始挖掘方块

eventobjid：挖掘方块的玩家迷你号

blockid：被挖掘的方块id

x,y,z：被挖掘方块的坐标

Block.Dig.Cancel

取消挖掘方块

eventobjid：取消挖掘方块的玩家迷你号

blockid：被取消挖掘的方块id

x,y,z：被取消挖掘的方块的坐标

Block.Dig.End

完成挖掘方块

eventobjid：完成挖掘的玩家迷你号

blockid：被完成挖掘的方块id

x,y,z：被完成挖掘的方块的坐标

Player.AreaOut

eventobjid：离开区域的玩家迷你号

Player.AddItem

itemnum：丢弃的物品数量
